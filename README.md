# 功能
- 地址栏全角字符自动转换功能：：转换成:   、转换成/
- 修复垂直标签状态下面当有固定标签时，右击关闭标签功能失效的bug
- 新标签打开收藏栏网站
- 右键标题栏打开新标签，并且粘贴剪贴板内容到地址栏
- 地址栏回车新开tab打开网页
- 右键关闭标签页
- 双击关闭标签页
- 保留最后标签页（防止关闭最后一个标签页时关闭浏览器，点X不行）
- 鼠标悬停标签栏滚动
- 按住右键时滚轮滚动标签栏
# 获取
下载edge_plus.zip
# 安装
dll放入Edge目录（C:\Program Files (x86)\Microsoft\Edge\Application）即可
